所有者信息

/claiminfo will show you information about a claim and allows to change a few claim specific settings.

Example output

Admin Settings

When you have control over claim a admin settings button will appear at the top of the claiminfo output.

DenyMessages

If set to OFF players won't see a warning message when trying to do things they don't have permissions/flags for.

ClaimExpiration

If set to OFF the claim won't be affected by claim expiration.

FlagOverrides

If set to OFF the claim ignores flag overrides for its claim type.

PvPOverride

Overrides the dimensions PvP settings from the Sponge config. It has three states:

TRUE - enables PvP in the claim
FALSE - disables PvP in the claim
UNDEFINED - uses the setting defined in the Sponge config
